# Pirate Tattoo Analysis

## About
This project uses R to analyse tattoo counts across pirate age groups using the `pirates` dataset from the `yarrr` package. The main output is a scatter plot comparing tattoo distribution by age range, with a separate script that exports summary statistics to CSV.

## Data
Data comes from the `pirates` dataset in the `yarrr` R package. No external files needed.
